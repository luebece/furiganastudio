자동 후리가나 & 단어장

일본어 문장을 넣으면 자동으로 후리가나(요미가나)를 달아줍니다. 모르는 단어를 클릭해 뜻을 확인하고 나만의 단어장을 만드세요.

로컬 AI 음성 합성

PC에 설치된 VOICEVOX 또는 AivisSpeech와 연동하여, 네트워크 지연 없이 고품질 캐릭터 음성으로 문장을 읽어줍니다.

살아 숨쉬는 Live2D

단순한 텍스트 뷰어가 아닙니다. 음성에 맞춘 립싱크, 마우스 트래킹, 클릭 상호작용 및 호감도 시스템이 학습의 재미를 더합니다.

단 3단계로 바로 시작하기

1

로컬 음성 엔진 켜기 (필수)

PC에 설치된 VOICEVOX 혹은 AivisSpeech를 먼저 실행해두어야 음성 합성 기능이 정상 작동합니다. 아직 설치하지 않았다면 아래 공식 사이트에서 받아주세요.

VOICEVOX 공식 사이트 AivisSpeech 공식 사이트
2

프로그램 다운로드

위의 메인 다운로드 버튼을 눌러 최신 버전의 FuriganaStudio.exe를 다운로드 받습니다.

3

설치 없이 즉시 실행

다운로드 받은 파일을 더블클릭하면 끝! 무거운 설치 과정 없이 독립적인 포터블 형태로 바로 실행됩니다.

Windows 보안 경고가 뜨나요?
개인 개발자가 배포하는 exe 파일은 윈도우 디펜더에서 경고를 띄울 수 있습니다. 안심하고 [추가 정보] → [실행]을 눌러주세요.
